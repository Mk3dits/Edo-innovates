Unit No. Formatting tags Description
1 b makes a selected text bold
2 i transforms a selected text into an italic writing
3 u underlines a selected text
4 strike strikes through a selected text
5 em used to display content in italic
6 mark used to highlight text
6 del displays deleted content
7 big to increase the font size by one conventional unit
8 small to decrease font size by one unit from base font size
9 tt used to display text in teletype
10 ins displays the tag that is added
11 sup displays content slightly above the normal line
12 sub displays content slightly below the normal line
